# typing-test
Тренажёр слепой печати

Для запуска приложения в терминале выполните

<p>git clone git@github.com:Arthur1995dev/typing-test.git<p>
<ul>
<li> Перейдите в папку проекта с помощью команды: cd typing-test </li>
<li> Далее для установки всех модулей и зависимостей: npm install </li>
<li> Запускаем проект: npm start </li>
</ul>
